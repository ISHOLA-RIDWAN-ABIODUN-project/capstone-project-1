About

I deployed this site using Amazon S3 for storage and CloudFront for global content delivery. It’s a simple project, but it demonstrates real cloud fundamentals: permissions, caching, and deployment flow.

Tech Stack

Amazon S3

Stores and serves the site assets.

CloudFront

Global CDN URL for fast delivery.

HTML + CSS

Static frontend with responsive layout.

Demo Notes

  1. Create S3 bucket and upload files.
  2. Enable website behavior and set index document.
  3. Create CloudFront distribution and set default root object.
  4. Verify live URL loads correctly.
